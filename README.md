# SnRNAseq_ASO_Myenteric_Plexus
[![DOI](https://zenodo.org/badge/982844916.svg)](https://doi.org/10.5281/zenodo.15793861)

Github repository associated with Pukszta et al. 2025 "Single-nuclei transcriptome of enteric neurons in Î±-synuclein overexpressing mice". Contains code to analyze single-cell RNA-seq data. Files are numbered in the order they were run to produce final results. Raw and post-processing data files are available on the Gene Expression Omnibus (GEO) at accession number 297092 (https://www.ncbi.nlm.nih.gov/geo/query/acc.cgi?acc=GSE297092).
